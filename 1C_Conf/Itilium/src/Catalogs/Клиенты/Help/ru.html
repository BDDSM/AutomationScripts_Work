Справочник «Клиенты»

Справочник предназначен для хранения перечня клиентов организации. Справочник реализован на форме, следующие закладки: «Общая информация», «Пользователи», «Соглашения», «Перечень услуг», «Конфигурационные единицы», «Интерфейс пользователя» и «Контактная информация».

На закладке «Общая информация» находятся общие реквизиты клиента, такие как:

  • Краткое наименование;
  • Название организации;
  • Контрагент;
  • Комментарий.

Элемент справочника «Клиенты» у которого установлен флаг «Склад» предназначен для хранения на нем единиц номенклатуры. К нему не могут быть привязаны рабочие места, клиенты и д.р. атрибуты элемента справочника клиенты.

Закладка «Пользователи» заполняется из подчиненного справочника «Пользователи». С помощью кнопки «Синхронизировать» можно объединить список пользователей Active Directory (если используется) и приложения. При нажатие на кнопку «Синхронизировать» запустится обработка загрузки пользователей из Active Directory. Загрузка выполняется в два этапа, сначала загружается иерархия Active Directory, затем выполняется загрузка выбранных пользователей. При установке галочки «Использовать дополнительные данные» загружаются дополнительные параметры учетной записи Active Directory (например: должность, контактная информация и комментарий).

На закладке «Соглашения» можно просмотреть список договоров, заключенных с этим клиентом.

На закладке «Перечень услуг» можно просмотреть перечень услуг, предоставляемый клиенту, по заключенным договорам. В справочнике «Клиенты» перечень услуг является информационным полем и редактированию не подлежит, для изменения перечня услуг, следует редактировать документ «Соглашение об уровне услуг».

На закладке «Конфигурационные единицы» можно просмотреть перечень конфигурационных единиц, на которые распространяются перечисленные услуги. В справочнике «Клиенты» перечень объектов обслуживания является информационным полем и редактированию не подлежит, для изменения перечня объектов обслуживания, следует редактировать документ «Соглашение об уровне услуг».

На закладке «Интерфейс пользователя» хранятся настройки форм регистрации обращений от пользователей данного клиента. Возможны следующие значения:

  • Использовать настройки по-умолчанию при регистрации обращения. Будут использоваться настройки, заданные в параметрах системы (обработка "Параметры системы").
  • Выбирать услугу, состав услуги при регистрации обращения.
  • Выбирать конфигурационную единицу при регистрации обращения.
  • Отображать список дополнительных полей при регистрации обращения.

На закладке «Контактная информация»  настраивается контактная информация Клиента - электронная почта, телефоны, адреса и т.п.

 

При заключении индивидуального Соглашения об уровне сервиса для Клиента с признаками "Холдинг" и "Юридическое лицо" соглашение заключается для указанного клиента и всех подразделений, принадлежащих данному клиенту.
